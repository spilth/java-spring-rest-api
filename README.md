# Java/Spring REST API Example

## Setup

### Java Install

```bash
$ brew cask install java
$ brew install maven
```

### Project Setup

```bash
$ git clone git@github.com:spilth/java-spring-rest-api.git
$ cd java-spring-rest-api
$ mvn spring-boot:run
```

Then visit <http://localhost:8080/api/v1/message>
